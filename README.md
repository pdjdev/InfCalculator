## Build
```
gcc -o inf_calc main.c
```

## Usage
```
inf_calc input
inf_calc input -log
inf_calc -type
inf_calc -type -log
```
### Options
`-log` : 수정된 후위 표기식, 계산 과정을 출력합니다

`-type` : 파일에서 식을 읽는 대신 키보드로 식을 바로 입력합니다
